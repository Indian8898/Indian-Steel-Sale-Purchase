Google Drive
Auto-sync saves your data to a hidden file in your Google Drive app data folder.

Important: Google Drive sign-in works only on your hosted HTTPS app, like GitHub Pages. It will not connect from the local file:// preview.

Step-by-step setup:
1. Open your hosted app URL, not the local file preview.
2. Go to console.cloud.google.com and create/select a project.
3. Enable Google Drive API for that project.
4. Open OAuth consent screen and complete the basic app details.
5. Open Credentials and create an OAuth Client ID for a Web application.
6. In Authorized JavaScript origins, add your GitHub Pages origin, for example:
https://indian8898.github.io
7. Copy the generated Client ID.
8. Paste that Client ID below and tap Save Client ID.
9. Tap Connect Google Drive and sign in to your Google account.
10. Tap Sync from Drive Now once to test.
